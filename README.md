# haptic car

### open in VSCode (using [PlatformIO](https://platformio.org/platformio-ide))

### Libraries used:
* [JMotor](https://github.com/joshua-8/JMotor) library for motor control
* [ESP32_easy_wifi_data](https://github.com/joshua-8/ESP32_easy_wifi_data) as method of communication
* [RCMv2](https://github.com/rcmgames/RCMv2) template
